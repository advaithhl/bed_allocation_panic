# bed allocation panic lol

ok so. you drag tiny fake patients into tiny fake rooms. if the rules say no, the game says no. if the queue gets spicy, your brain might also say no. that’s pretty much it.

it’s kinda “hospital logistics” themed except nobody’s doing real medicine and the names are unhinged on purpose.

**play:** drag people places. panic. move them again when you painted yourself into a corner. they eventually leave so beds open up. there’s a tutorial the first time; click through when you’re ready.

**run it:**

```bash
npm install
npm run dev
```

whatever url vite yells at you — usually localhost something.

`npm run build` / `npm run preview` / `npm run lint` exist if you care.

**tech stuff (sorry):** react, typescript, vite, tailwind, zustand, dnd-kit, framer motion. if you wanna crank the numbers (spawn speed, how mean the game is, etc.) poke `src/config.ts`.

**the vibe-coded part:** this repo is basically hours of “yeah that sounds fun” in a cursor window. started from a messy requirements doc, then kept stacking features until it felt like an actual game. lots of it is ai-ish pair programming — you tweak a sentence, the editor hallucinates a feature, you go “actually no” and repeat. this readme got the same treatment: not sacred, not corporate, just here so future-you remembers what’s going on.

playground project — do whatever with it.
